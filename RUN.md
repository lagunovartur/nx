###  Создаем sim link на ./cli/run в /usr/local/bin
```shell
sudo ./cli/run in-link
```

### Запуск проекта
```shell
nx dc up -d
```

### Применяем миграции
```shell
nx al-up
```

### Стоп проекта
```shell
nx dc down
```
